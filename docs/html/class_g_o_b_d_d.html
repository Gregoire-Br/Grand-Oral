◆ userQuery() [1/3]

userQuery ( string  $user)

Cherche un utilisateur et retourne les informations sur lui.

Parameters
user- Nom d'utilisateur à chercher
Returns
rslt - tableau associatif contenant toutes les informations, avec ces paires :
  • username: nom d'utilisateur
  • password: mot de passe hashé
  • firstname: prénom
  • lastname: nom de famille
  • status: grade de l'utilisateur; 0: étudiant; 1: enseignant; 2: proviseur adjoint; 3: secrétaire
  • email: adresse email
